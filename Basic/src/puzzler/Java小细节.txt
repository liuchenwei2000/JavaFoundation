                             Java 小细节

path是告诉操作系统Java命令的存放位置(即执行路径：指操作系统搜寻本地可执行文件的目录列表)，值为%JAVA_HOME%\bin。
classpath是告诉Java编译器去哪里查找第三方类库和自定义类库。

若把int型强制转换成char型，将得到int值对应的ASCII码所代表的数字字母和符号。

Java的char是采用Unicode字符集UTF-16编码，每个占2Byte。

科学计数法的8.2*10^8在程序中应写为8.2e+8，8.2*10^(-8)应写为8.2e-8。

非简洁与(&)、非简洁或(|)将对其前后的逻辑运算都检查一遍然后返回结果。
简洁与(&&)、简洁或(||)对其前后的逻辑运算检查时一旦能够确定结果即返回不再检查剩下的逻辑运算。
 
基类的构造函数既不能被子类继承也不能被覆盖。

若某个类只有一个构造函数，并且是private的，这意味着：
1，强制此类外不可创建其实例  2，此类不可被继承

import 包名.类名；//导入这个包的单一类
import 包名.*;//导入这个包下的所有public类，但是不导入该包下的子package中的类

int a = 2;//创建一个变量a，它占有一小块内存，里面放入2(基本类型)
Object ob;//创建一个变量ob，它占有一小块内存，里面放入ob(引用)
ob = new Object();//在内存中创建一个Object对象，ob这个引用指向内存中的这个对象

final修饰的变量只能在第一次初始化(定义时或者构造方法中)时被赋值。

系统路径作为参数时的表示方法是把"\"都转换成"/"

尽量不要在循环中构造和释放对象(为了节省内存和性能优化)

尽量不使用l(小写的L)作为变量名，因为容易和数字1混淆，并且在表示long型整数时，使用L作为后缀标识，而不是小写的L(l)

当null被赋给一个对象引用之后，这时打印这个引用的话会自动转换成"null"
