                                                                                                            嵌套类和多重继承

使用嵌套类最吸引人的原因：
每个嵌套类都能独立的继承自一个（接口的）实现，所以无论外部类是否已经继承了某个(接口的)实现，对于嵌套类都没有影响。
嵌套类允许继承多个非接口类型（类或抽象类）这样有效地支持了多重继承。

如：
若必须在一个类中实现两个接口，可有两种选择：
interface A{}
interface B{}
1，使用单一类实现这两个接口。
class X implements A,B{}
2，使用嵌套类
class Y implements A {
		B makeB() {
			return new B() {
			};
		}
	}
若必须在一个类中实现一个类和一个抽象类则必须使用嵌套类实现多重继承：
class D{}
abstract E{}
class Z extends D{
E makeE() {
			return new E() {
			};
		}
	}
